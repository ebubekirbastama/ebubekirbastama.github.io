HTML Temizleyici

Bu araç, HTML kodlarınızı temizlemek ve gereksiz öznitelikleri kaldırmak için kullanabileceğiniz pratik bir çözümdür. HTML kodlarınızdaki class, id ve stil özniteliklerini temizler, böylece daha sade ve okunabilir bir kod yapısı elde edersiniz. Ayrıca, iframe öğelerine dokunmaz, bu sayede gömülü içerikleriniz güvende kalır.

Nasıl Kullanılır?

Aracı kullanmak oldukça basittir. Aşağıdaki adımları izleyerek HTML kodlarınızı temizleyebilirsiniz:

  1. HTML kodunuzu aşağıdaki ilk metin kutusuna yapıştırın.
  2. "Temizle" düğmesine tıklayın.
  3. Temizlenmiş HTML kodu ikinci metin kutusunda görünecektir.

Temizlenen Öğeler

Bu araç, HTML kodlarınızdaki şu öznitelikleri temizler:

  • class - HTML öğelerinin stilini belirleyen öznitelik.
  • id - HTML öğelerini tanımlamak için kullanılan benzersiz öznitelik.
  • style - HTML öğelerine doğrudan stil ekleyen öznitelik.

Dokunulmadan Bırakılan Öğeler

Bu araç, iframe öğelerine dokunmaz. Bu sayede, sayfanızdaki gömülü içerikler ve videolar etkilenmez.
